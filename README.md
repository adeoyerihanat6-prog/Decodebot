# 🤖 DecodeBot

DecodeBot is a simple rule-based AI chatbot built with **HTML, CSS, and JavaScript**. It simulates a conversation by recognizing predefined keywords and responding with appropriate messages.

## ⚙️ How It Works

1. The user types a message.
2. JavaScript converts the message to lowercase.
3. The chatbot checks for matching keywords using `includes()`.
4. If a match is found, an appropriate response is generated.
5. If no match is found, the chatbot returns a default response.
6. The conversation is displayed dynamically using DOM manipulation.
